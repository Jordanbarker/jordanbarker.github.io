Discrete Distributions ¶

Bernoulli Distribution ¶

Bernoulli(p) Binary, boolean output: 1 (success) with probability p, or 0 (failure) with probability q=1-p.

  • E[X] = p
  • Var(X) = p * q
In [23]:
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt
import scipy.stats as stats

p = 0.7 # probability of getting a "success" (heads)
num_simulations = 1000

# generate random bernoulli distributed variables (0 or 1)
simulations = np.random.binomial(n=1, p=p, size=num_simulations)

# count the number of successes and failures
successes = np.count_nonzero(simulations == 1)
failures = np.count_nonzero(simulations == 0)
print(f'Successes: {successes}, Failures: {failures}')

labels = ['Successes', 'Failures']
counts = [successes, failures]

plt.figure(figsize=(4, 2))
sns.barplot(x=labels, y=counts, palette=['#1f77b4', '#ff7f0e'])
plt.xlabel('Outcome')
plt.ylabel('Count')
plt.title('Bernoulli Distribution Simulation')
plt.show()

Binomial Distribution ¶

Parameters:

  • n: # of successes in a sequence of independent experiments, each asking a yesno question
  • p: probability of success or failure q=1-p

Properties:

  • E[Y] = n * p
  • Var(Y) = n * p * q

Geometric Distribution ¶

Number of Bernoulli trials until success.

Properties:

  • E[X]=1/p
  • Var(X)=q/p^2
