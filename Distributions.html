Bernoulli Distribution ¶

Bernoulli(p) Binary, boolean output: 1 (success) with probability p, or 0 (failure) with probability q=1-p.

  • E[X] = p
  • Var(X) = pq
In [17]:
import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt

p = 0.7 # probability of getting a "success" (heads)
num_trials = 1000

# generate random bernoulli distributed variables (0 or 1)
trials = np.random.binomial(n=1, p=p, size=num_trials)

# count the number of successes and failures
successes = np.count_nonzero(trials == 1)
failures = np.count_nonzero(trials == 0)
print(f'Successes: {successes}, Failures: {failures}')

labels = ['Successes', 'Failures']
counts = [successes, failures]

plt.figure(figsize=(4, 2))
sns.barplot(x=labels, y=counts, palette=['#1f77b4', '#ff7f0e'])
plt.xlabel('Outcome')
plt.ylabel('Count')
plt.title('Bernoulli Distribution Simulation')
plt.show()

Geometric Distribution ¶

f(x)=(1-p)k-1p | F(x)=1-(1-p)k where x=1,2,.. # of Bern trials until success, FFFS implies x=4 and p is the success probability. E[X]=1/p, Var(X)=q/p^2

Poisson Distribution ¶

Poisson(ƛ)

f(x)=ƛxe-ƛ/x! F(x)=e-ƛ ∑xi=0 ƛi/i!

E[X] =ƛ=Var(X)

Eg. Pois(12), What is the prob of exactly 8? (use pmf) 128*e-12/8! =6.55%
